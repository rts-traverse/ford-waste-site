# Ford Waste — Website

Public marketing website for **Ford Waste**, a waste-hauling / junk-removal / roll-off SMB in
**Greenville, SC** (the Upstate). Built by Traverse.

## Stack
- **Static HTML + Tailwind** (matches `chief-security-solutions-site` and `la-vida-landscapes-site`).
- Deploys on **Vercel**; domain via **Hover** (staged DNS cutover at launch).
- Visual system defined in **`DESIGN.md`** (generated by the design pass) — read it before any
  UI/visual change.

## Pages (planned)
Home · Services (+ per-service) · Service Area · **Get a Quote** (primary conversion) · About ·
Reviews · Contact.

## Conversion model
Persistent **"Get a Quote"** CTA + **click-to-call / click-to-text** + a **photo-upload quote
form**. Trust signals: Licensed & Insured / BBB / real reviews / named service area.

## Status
Baseline scaffold (M1). Content, brand assets, and domain are pending from the client (Jordan
Ford). Requirements + plan tracked in the Traverse Product Factory job
`clients/ford-waste/factory/20260530-ford-waste-website.md`.
